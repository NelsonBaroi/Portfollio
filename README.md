# Portfollio — Nelson Baroi

Personal portfolio website showcasing my professional journey, projects, and philosophy. Built with HTML, CSS, JavaScript, and powered by an AI twin chatbot.

## Sections

- **Home** — About me, career, and background
- **Chat** — AI twin that answers questions about my experience
- **Projects** — Coding and professional projects
- **Courses** — Certifications and learning journey
- **Philosophy** — Personal principles and mindset
- **CV** — Detailed resume

## Tech Stack

- HTML5, CSS3, JavaScript (vanilla)
- GSAP for animations
- Python (Flask) backend for chat API
- Hosted at [nbaroi.com](https://nbaroi.com)

## Setup

```bash
# Clone and open
open index.html
```
